# Double Wishbone Suspension System - 3D CAD Model

This repository contains the complete 3D computer-aided design (CAD) model of a classic independent double wishbone (A-arm) suspension mechanism. The assembly is modeled to evaluate kinematic behavior, suspension geometry constraints, and mounting point clearances.

## Model Specifications
- **Software used:** Onshape (Cloud-native CAD)
- **Export format:** STEP (.step) for cross-platform CAD interoperability

## Design & Components Feature Tree
- **Upper & Lower Control Arms (A-Arms):** Geometrically optimized to handle and analyze the vertical displacements.
- **Steering Knuckle / Upright:** Designed with appropriate mounting points for the control arms and wheel hub.

## How to View and Use This Model
1. Download the `Double_Wishbone_Suspension_Assembly.step` file from this repository.
2. Import the file into any mechanical design software (Onshape, SolidWorks, CATIA, Fusion 360, etc.) to analyze the linkages and geometric constraints.
